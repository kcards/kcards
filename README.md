# Overview

A web application to automatically moderate discussions.

# Setup

## Requirements

* Python 3.6+

## Installation

Install from source:

```sh
$ git clone https://github.com/kcards/kcards.git
$ cd kcards
$ make install
```

# Usage

Run the application:

```sh
$ make run
```
