# Overview

A K-Cards web application for moderating Q&A sessions easily.

# Setup

## Requirements

* Python 3.5+

## Installation

Install from source:

```sh
$ git clone https://github.com/kcards/kcards.git
$ cd kcards
$ make install
```

# Usage

Run the application:

```sh
$ make run
```
